About This Server

A custom-built TCP web server crafted with C# and .NET

🔌

Raw TCP Sockets

Built from the ground up using .NET's TcpListener and NetworkStream for maximum control and performance.

🌐

HTTP Protocol

Implements HTTP/1.1 protocol with proper headers, status codes, and content negotiation.

⚡

Multi-threading

Each client connection is handled in a separate thread for concurrent request processing.

📁

Static Files

Serves HTML, CSS, and JavaScript files with proper MIME types and caching headers.

🔒

Security

Includes path traversal protection and file extension validation for secure serving.

🎨

Custom Errors

Beautiful error pages with proper HTTP status codes (404, 403, 405, 400).

Technical Stack

💻 C#
⚙️ .NET 8
🔌 TCP Sockets
🌐 HTTP/1.1
